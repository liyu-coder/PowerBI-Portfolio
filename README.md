# IT Branch Monitoring Dashboard

## Project Overview

This project demonstrates an interactive Power BI dashboard designed to monitor the operational performance of multiple branches. The dashboard provides managers with real-time insights into uptime, incidents, and average response times.

## Business Problem

Organizations often collect operational data from multiple branches, but the information is difficult to analyze quickly. Managers need a dashboard that highlights key performance indicators and identifies branches requiring attention.

## Solution

A Power BI dashboard was developed using a CSV dataset containing 90 days of operational data from five branches.

## Tools Used

* Power BI Desktop
* CSV Dataset
* DAX Measures
* Microsoft Excel

## Dashboard Features

* KPI Card for Uptime Percentage
* Bar Chart showing Incidents by Branch
* Line Chart showing Average Response Time over Time
* Interactive filtering by date and branch

## Key Metrics

* Uptime Percentage
* Total Incidents
* Average Response Time

## Business Value

The dashboard enables managers to monitor branch performance, identify operational issues, and support faster, data-driven decision-making.
